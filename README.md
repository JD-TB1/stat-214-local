# STAT 214 Local Workspace

This repository is the local working copy for STAT 214 labs.

## Layout
- `lab1/`: PECARN TBI analysis
- `lab2/`: MISR cloud-detection analysis

## Organization rules
- Each lab should keep its own code, documents, and generated outputs under its own directory.
- There should be no top-level generated artifact folders outside the lab directories.
- Shared generated outputs should live under each lab's own output conventions, not at the repository root.

## Current conventions
- `lab1/` uses the older `code/`, `scripts/`, `documents/`, `output/`, and `report/` structure.
- `lab2/` uses the newer `code/`, `documents/`, and `results/` structure split by assignment part.
